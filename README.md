## Setup

This project uses [uv](https://github.com/astral-sh/uv) for Python package management.

```bash
# Create virtual environment
uv sync

# Activate virtual environment
source .venv/bin/activate

# Run solutions
python bioinformatics-stronghold/dna.py
```

- Python >= 3.13
